## Sampling Rate

- **After every task commit:** Run quick command（conversation handler 单测）
- **After every plan wave:** Run `make test`
- **Before `/gsd:verify-work`:** `make verify`（lint/typecheck/test/smoke）must be green
- **Max feedback latency:** ~15 seconds（quick）
