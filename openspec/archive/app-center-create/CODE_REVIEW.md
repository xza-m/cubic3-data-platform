**问题**:

- ❌ 缺少前端单元测试（Jest/Vitest）

```bash
  integration/
    test_api.py
```

### 2. 异步执行实现 (优先级：中)

**问题**:
- ⚠️ 当前使用 `threading` 模拟异步执行
- ⚠️ RQ Worker 配置已就绪但未实际使用

**建议**:
```python

queue = Queue(connection=redis_conn)

```

### 3. 配置验证过于严格 (优先级：低)

**问题**:
- ⚠️ 创建实例时强制验证 datasource_id 和 dataset_id 存在
- ⚠️ 开发环境测试不便

**建议**:
- 提供 `skip_validation` 选项用于开发/测试
- 或者提供 Mock 执行器

### 4. API 更新方法缺失 (优先级：低)

**问题**:
- ❌ `PATCH /app-instances/{id}/status` 端点未实现
- ❌ 实例启用/禁用只能通过 `PUT` 完整更新

**建议**:

### 5. 前端错误处理 (优先级：低)

**问题**:

- ⚠️ 网络错误提示不够友好
