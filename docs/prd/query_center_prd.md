**关键交互元素：**

1. **顶部工具栏**
   - 数据源切换（下拉选择）
   - 多Tab支持（像浏览器一样）
   - 快捷操作按钮

2. **三栏布局**
   - 左侧：SQL编辑器（Monaco Editor）
   - 右侧：数据库结构树
   - 底部：结果展示区域

3. **可调整分隔线**
   - 编辑区和结果区之间可拖拽调整高度
   - 侧边栏可折叠

**交互特点：**
- **步骤式引导**：清晰的1-5步流程
- **实时预览**：每次修改即时显示SQL
- **快捷操作**：常用时间范围（今天、本周、上月）
