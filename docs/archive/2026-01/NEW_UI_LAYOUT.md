## 📋 功能模块详细说明

### 模块 1: 📊 Superset 订阅

#### Tab 1: 任务列表
- **功能**: 管理 Superset 看板推送任务
- **操作**: 创建任务、编辑任务、删除任务、手动执行
- **显示**: 任务列表表格（ID、名称、Cron、状态等）

#### Tab 2: 群列表管理
- **功能**: 管理飞书群组
- **操作**: 启用/禁用群组、刷新群列表
- **显示**: 群组列表表格（chat_id、名称、状态等）

#### Tab 3: 任务运行记录
- **功能**: 查看任务执行历史
- **操作**: 刷新记录
- **显示**: 运行日志表格（任务ID、状态、耗时、错误等）

---

### 模块 2: 📥 数据导出

#### Tab 1: 查询导出 ⭐
- **功能**: 自助数据查询和导出
- **步骤**:
  1. 选择数据集（下拉选择）
  2. 选择导出字段（多选，分类展示：分区键/维度/度量）
  3. 设置筛选条件（动态添加）
  4. 配置高级选项（行数限制、输出格式）
  5. 提交导出任务
- **特色功能**:
  - ✅ 敏感字段自动标记（🔒脱敏）
  - ✅ SQL 预览功能
  - ✅ 实时任务状态监控
  - ✅ 下载链接自动生成

#### Tab 2: 导出历史
- **功能**: 查看历史导出任务
- **显示**: 任务历史表格
  - 任务ID
  - 数据集名称
  - 状态（成功/失败）
  - 导出行数
  - 文件大小
  - 提交时间
  - 耗时
  - 下载按钮
- **操作**: 刷新列表、下载文件

#### Tab 3: 数据集管理
- **功能**: 浏览可访问的数据集
- **展示方式**: 卡片式布局
- **卡片信息**:
  - 数据集名称
  - 敏感级别标签（PUBLIC/INTERNAL/CONFIDENTIAL）
  - 数据集描述
  - 数据域
  - 字段数量
  - 数据集代码
- **交互**: 点击卡片快速跳转到查询导出页面并选中该数据集

## 🎯 导航交互逻辑

### 主模块切换
```javascript
// 点击左侧导航项
switchModule('superset')  // 切换到 Superset 订阅模块
switchModule('export')    // 切换到数据导出模块
```

**效果**:
- 隐藏其他模块页面
- 显示选中模块页面
- 更新左侧导航高亮状态
- 自动加载该模块的默认数据

### 子 Tab 切换
```javascript
// 在 Superset 模块内切换 Tab
switchSubTab('superset', 'tasks')  // 任务列表
switchSubTab('superset', 'chats')  // 群列表管理
switchSubTab('superset', 'logs')   // 任务运行记录

// 在数据导出模块内切换 Tab
switchSubTab('export', 'query')    // 查询导出
switchSubTab('export', 'history')  // 导出历史
switchSubTab('export', 'datasets') // 数据集管理
```

**效果**:
- 仅在当前模块内切换 Tab
- 不影响其他模块
- 按需加载数据

## 🔄 与旧版的对比

### ❌ 旧版（单层 Tab）
```
[任务列表] [群列表] [运行记录] [数据导出]
```
**问题**:
- 所有功能平铺在一级
- 功能多时 Tab 过于拥挤
- 难以扩展新模块

### ✅ 新版（两级导航）
```
左侧主导航:
  📊 Superset 订阅
  📥 数据导出

右侧子功能:
  Superset: [任务列表] [群列表] [运行记录]
  数据导出: [查询导出] [导出历史] [数据集管理]
```
**优势**:
- ✅ 层次清晰，易于理解
- ✅ 模块独立，易于扩展
- ✅ 功能分组，减少认知负担
- ✅ 支持未来添加更多主模块
