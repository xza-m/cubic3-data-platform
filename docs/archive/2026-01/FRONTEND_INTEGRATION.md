async function pollTaskStatus(taskId) {
    const interval = setInterval(async () => {
        const res = await fetch(`${API_BASE}/api/v1/export/status/${taskId}`);
        const task = await res.json();
        showExportTaskStatus(task);
        
        if (task.status === 'SUCCESS' || task.status === 'FAILED') {
            clearInterval(interval);
        }
    }, 2000);
}

## 🐛 故障排查

### 问题 1: 看不到"数据导出"Tab
**原因**: 浏览器缓存未更新

**解决方案**:
```bash
# 强制刷新浏览器
Ctrl + F5 (Windows/Linux)
Cmd + Shift + R (macOS)
```

### 问题 2: 点击 Tab 无反应
**原因**: JavaScript 未正确加载

**解决方案**:
```bash
# 检查浏览器控制台是否有错误
F12 → Console 标签
```

### 问题 3: Mock 数据未显示
**原因**: 数据集加载函数未执行

**解决方案**:
```javascript
// 在浏览器控制台手动调用
loadDatasets();
```
