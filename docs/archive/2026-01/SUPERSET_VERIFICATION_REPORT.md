## 📝 技术细节

### 页面架构
```
superset_new.html (extends console_base.html)
├── extra_css block
│   └── 自定义样式 (按钮、卡片、模态框等)
├── content block
│   ├── 页面头部 (标题 + 操作按钮)
│   ├── 统计卡片 (今日推送、活跃任务、平均耗时)
│   ├── 任务卡片网格
│   ├── 任务模态框 (创建/编辑)
│   ├── 群聊模态框 (管理群列表)
│   └── 日志模态框 (查看历史)
└── extra_js block
    └── 所有JavaScript逻辑
```

### API依赖
```
/api/tasks              - 任务CRUD
/api/tasks/<id>         - 单个任务操作
/api/tasks/<id>/run     - 手动执行
/api/tasks/runs         - 运行记录
/api/feishu/chats       - 活跃群聊
/api/feishu/chats/all   - 所有群聊
/api/feishu/chats/<id>  - 群聊操作
```

### 数据流
```
页面加载
  ↓
init()
  ├── loadChats(true)  → /api/feishu/chats
  ├── loadTasks()      → /api/tasks
  └── loadStats()      → /api/tasks/runs + /api/tasks
      ↓
渲染UI
  ├── 统计卡片
  ├── 任务卡片
  └── 下拉选项
```
