## 📋 三步注册流程

### 步骤 1: 输入表名

**界面元素**:
- 智能识别说明卡片（置信度说明）
- MaxCompute 项目名输入框
- 表名输入框
- "覆盖已存在的数据集"复选框
- 🔌 测试连接按钮
- 👁️ 预览识别结果按钮

**操作流程**:
```
1. 输入项目名：prod_dw
2. 输入表名：dwd_trade_order_detail
3. (可选) 点击"测试连接"验证表是否存在
4. 点击"预览识别结果"
```

### 步骤 2: 预览识别结果

**界面元素**:
- 表信息卡片（表名、注释、数据量、字段数）
- 识别统计卡片
  - 分区字段数（紫色）
  - 度量字段数（蓝色）
  - 敏感字段数（红色）
  - 总字段数（灰色）
- 字段列表表格
  - 物理字段名
  - 业务名称（可编辑）
  - 类型
  - 类别（分区/度量/维度）
  - 敏感标记
  - 置信度进度条
  - 编辑按钮

**操作流程**:
```
1. 查看识别结果
2. (可选) 双击业务名称进行修改
3. (可选) 点击"编辑"按钮修改字段属性
4. (可选) 点击"导出 CSV"保存识别结果
5. 确认无误后，点击"确认并注册"
6. (可选) 点击"返回修改"重新设置
```

### 步骤 3: 注册完成

**界面元素**:
- 🎉 成功动画
- 注册结果卡片
  - 数据集ID
  - 批次ID
  - 同步字段数
  - 耗时
- "注册另一个数据集"按钮
- "开始使用"按钮

**操作流程**:
```
1. 查看注册结果
2. 点击"注册另一个数据集"继续注册
   或
   点击"开始使用"跳转到查询导出页面
```

## 💡 使用技巧

### 1. 业务名称快速编辑

字段表格中的"业务名称"列支持直接编辑：
```
双击 → 编辑 → 失焦自动保存
```

### 2. 批量导出识别结果

点击"导出 CSV"按钮，可以将识别结果导出为 CSV 文件：
```csv
物理字段名,业务名称,类型,类别,敏感,置信度
ds,数据日期,STRING,PARTITION_KEY,false,1.0
user_mobile,手机号,STRING,DIMENSION,true,0.95
```

### 3. 测试连接功能

在预览之前，建议先点击"测试连接"：
- ✅ 验证表是否存在
- ✅ 验证是否有访问权限
- ✅ 避免后续步骤失败

### 4. 置信度参考

- **1.0 (100%)** - 分区字段，完全准确
- **0.9+ (90%+)** - 置信度很高，通常正确
- **0.7-0.9 (70%-90%)** - 置信度较高，建议复核
- **<0.7 (<70%)** - 置信度较低，强烈建议人工确认

### API 2: 确认注册

```javascript
const res = await fetch('/api/v1/metadata/sync/trigger', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-User-ID': 'current_user'
  },
  body: JSON.stringify({
    project: 'prod_dw',
    table: 'dwd_trade_order_detail',
    override_existing: false
  })
});
const result = await res.json();
```

## 📝 注意事项

### ⚠️ Mock 数据模式

当前前端使用 Mock 数据进行演示。要启用真实 API：

```javascript
// 在 JavaScript 代码中找到以下注释
// TODO: 实际调用 API

// 取消注释真实 API 调用代码
const res = await fetch(`${API_BASE}/api/v1/metadata/sync/preview...`);

// 删除或注释掉 Mock 数据代码
// Mock 数据
// const data = { ... };
```

### ⚠️ MaxCompute 连接

确保后端已配置 MaxCompute 连接：

```bash
# .env
MC_ACCESS_ID=your_access_id
MC_SECRET_KEY=your_secret_key
MC_ENDPOINT=http://service.odps.aliyun.com/api
```
