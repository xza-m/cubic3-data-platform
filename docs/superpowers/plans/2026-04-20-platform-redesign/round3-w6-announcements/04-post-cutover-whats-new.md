## 焕新一目了然（5 bullet）

- **统一信息架构**：五大模块（数据中心 / 查询 / 语义 / 应用市场 / 配置）按工作流重排，导航与面包屑一致；
- **更快的首屏**：v2 首屏 gzip 包 ≤ 350 KB（实测 283.8 KB），关键路径冷启动 < 2.5 s；
- **可观测的故障**：错误上报与埋点 dashboard 上线，平均故障定位时间 ≤ 5 分钟；
- **更稳的视觉**：Round 3 视觉基线刷新，5 张关键页面 axe a11y 0 严重违规；
- **平滑迁移**：旧 URL 自动 301 到新路径，6 个月内书签仍可用。

## 功能改进列表（按模块）

### 数据中心

- 数据源：列表支持按状态 / 类型筛选；详情页"测试连接"按钮直接调用后端，结果含延迟与版本；
- 数据集：注册流程拆为"按表 / 按文件"两条；字段画像与样例值在详情页直读；
- 提取任务：调度配置内嵌于详情；执行历史按状态色分组。

### 查询

- 已保存查询、查询历史、计划任务三栏并列；
- 查询历史筛选支持按状态 / 数据源 / 执行人组合；
- 计划任务详情新增"手动 run"按钮（沿用 Round 2 后端能力）。

### 语义

- 本体工作台二级导航（工作台 / 对象 / 指标 / 关系 / 治理）统一；
- Cube 列表 + 详情 + 编辑形成完整 lifecycle，新增 Join 关系画布；
- 指标行内"预览"（dry-run）入口，支持 SQL 即时预览。

### 应用市场

- 应用按 facet 筛选（已上架 / 待审核 / 已下架）；
- 实例列表新增 health chip，故障一眼可见；
- 实例详情接入实例级 health 接口（B-back-9）。

### 配置中心

- 渠道、订阅、用户、角色四栏并列；
- 渠道详情新增"试发送"动作；订阅详情新增执行历史标签页；
- 角色管理上线（P14），权限矩阵可视化。
