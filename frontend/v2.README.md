# frontend/v2.README.md — v2 Preview 启动指南

## 开发预览

v2 使用独立 Vite 配置，**不需要修改 package.json**。

  ```sh
  cd frontend
  npx vite --config v2.vite.config.ts
  ```

默认监听 **http://localhost:3001**，API 请求代理到 `http://localhost:81`。

## 环境变量

| 变量 | 说明 |
| --- | --- |
| `VITE_API_PROXY_TARGET` | 后端地址（默认 `http://localhost:81`） |
| `VITE_AUTH_BYPASS=1` | 跳过 JWT 认证（**仅本地开发**） |

## 生产构建

  ```sh
  cd frontend
  npx vite build --config v2.vite.config.ts
  # 产物输出到 frontend/dist-v2/
  ```

## 路径别名

| 别名 | 解析路径 |
| --- | --- |
| `@v2` | `frontend/src/v2/` |

## 与 legacy 的隔离

- legacy 使用 `vite.config.ts`（port 3000，alias `@`）
- v2 使用 `v2.vite.config.ts`（port 3001，alias `@v2`）
- 两者**完全隔离**，不互相 import
