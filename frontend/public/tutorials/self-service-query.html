自助查询入门

本教程带你从数据目录选择数据源和表，理解字段信息，编写 SQL，运行查询，并把常用查询保存到“我的查询”。完成后，你可以独立完成一次受控的数据探索。

15 min Beginner 查询中心

学习目标

建立查询工作台的基础操作心智。

1

选择资源

在左侧资源目录中先选择数据源，再选择数据库。表列表会分页展示，悬浮表名可以查看描述，点击表名后查看字段、类型和业务说明。

2

编写并运行 SQL

把表名填入编辑器，先用小 LIMIT 验证字段和过滤条件，再逐步扩展查询。结果区默认每页 20 条，便于横向扫字段。

SELECT *
FROM df_cb_258187.ads_bi_question_base_stats_df
WHERE ds = '20260506'
LIMIT 20;
3

保存与复用

确认 SQL 可重复执行后保存到“我的查询”。保存后的查询可以直接打开到工作台继续编辑，不应跳转到空白页。

完成检查

用下面三个信号判断你已经掌握查询工作台。

✓

能够从目录定位表

知道数据源、数据库和表之间的层级关系，并能查看字段说明。

✓

能够解释结果

能说明每个查询字段的含义，知道是否需要分区过滤、时间过滤或聚合口径。

✓

能够保存复用

保存后的 SQL 在“我的查询”里可检索、可打开、可继续编辑。
